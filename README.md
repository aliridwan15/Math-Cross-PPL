## ğŸ“– Tentang MathKids

**MathKids** adalah game edukasi berbasis *puzzle* matematika yang dirancang khusus untuk membantu anak-anak dan pelajar belajar berhitung dengan cara yang menyenangkan.

Dalam permainan ini, pemain ditantang untuk menyelesaikan berbagai soal matematika (penjumlahan, pengurangan, perkalian, pembagian) yang disajikan dalam format **Teka-Teki Silang Interaktif**. Tujuannya adalah mengubah persepsi matematika yang "sulit" menjadi permainan yang adiktif dan seru.

### âœ¨ Fitur Utama
* **Mode Belajar:** Tutorial interaktif untuk pemain baru.
* **Sistem Level:** Terbagi menjadi tingkat kesulitan *Mudah*, *Sedang*, dan *Sulit*.
* **Progressive Chapters:** Terdapat 5 Chapter di setiap tingkat kesulitan.
* **Scoring System:** Penilaian skor hasil di akhir permainan.
* **Interactive UI:** Tampilan yang ramah anak dan mudah digunakan.

## ğŸš€ Cara Menjalankan Project

1.  **Clone Repository**
    ```bash
    git clone [https://github.com/aliridwan15/Math-Cross-PPL.git](https://github.com/aliridwan15/Math-Cross-PPL.git)
    ```
2.  Buka **Unity Hub**.
3.  Klik **Add** -> **Add project from disk**.
4.  Pilih folder hasil clone.
5.  Pastikan menggunakan Editor Version **6000.0.2.72** (atau versi yang kompatibel).
6.  Buka Scene `HomeScreen` atau `MainMenu` untuk memulai.
7.  Tekan tombol **Play**.
